# qmd Local Search for Markdown Vaults

## Source Snapshot

- URL: https://github.com/tobi/qmd
- Type: repo

## Repository Notes

qmd is a local markdown search tool oriented around searching notes on disk without immediately introducing a hosted vector stack.

The repository matters in this vault because it represents a concrete local-search upgrade path between plain markdown navigation and full RAG infrastructure.

It should be treated as evidence about local-first retrieval strategy, not as a reason to jump straight to embeddings.
