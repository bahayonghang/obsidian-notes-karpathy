# obsidian-notes-karpathy

Use the package entry skill when the user describes the workflow as a whole or asks what step should run next.

## Main routing outcomes

- `kb-init` for fresh, partial, or legacy V1 vaults
- `kb-compile` when raw captures are ahead of drafts
- `kb-review` when drafts are pending or briefings are stale
- `kb-query` when the live layer is ready for retrieval
- `kb-health` when approved knowledge needs maintenance
